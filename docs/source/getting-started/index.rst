Getting Started
=========================

This guide covers the installation and basic usage of ChromaQuant and its submodules.

Installation
-------------------------

To start using ChromaQuant, first install the package using pip:

.. code-block:: bash
    
    pip install chromaquant

You can then verify your installation using the following:

.. code-block:: bash

    pip show chromaquant
