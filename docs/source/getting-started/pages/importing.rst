Importing ChromaQuant
----------------------

ChromaQuant can be imported into a Python script like so:

.. code-block:: python
    
    import chromaquant as cq

Note the use of :code:`cq` as an alias for :code:`chromaquant`. Although users are welcome to give
their imports any alias they like (or none at all), all documentation moving forward will assume an
alias of :code:`cq`. Please keep this in mind if you decide to customize your alias.
